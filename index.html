Современные изменения климата района исследования

Материалы и методы

Для анализа современных климатических изменений района исследований были использованы данные измерений приземной температуры воздуха и количества осадков на метеостанции п. Бахта (62°27′48″ с. ш. 89°00′15″ в. д.), расположенной в 20 км к северу от Енисейской экологической станции “Мирное” ИПЭЭ РАН. Данные за период 01.01.1961-31.01.2005 в суточном разрешении были предоставлены ФГБУ “ВНИИГМИ-МЦД”. Данные за период 01.02.2005-31.05.2023 были загружены из архива https://rp5.ru/ (одно измерение в 3 часа, было осреднено до суточного разрешения). Единичные пропущенные значения были заменены методом линейной интерполяции (функция na_interpolation из пакета imputeTS). Для расчета экстремальных (для мелких млекопитающих) периодов в дальнейшем были использованы очищенные данные суточного разрешения.

Для расчета климатических норм и отклонений от них данные были осреднены до месячного разрешения. В период с 1966 по 1976 годы были обнаружены некорректные значения температур приземного воздуха в летне-осенний период (июнь, июль, август, сентябрь). Эти значения были заменены на осредненные за тот же период по данным ближайших метеостанций п. Бор (61°35′56″ с. ш. 90°00′41″ в. д.; 112/95 км на ю-в от п. Бахта/стационара “Мирное” ) и с. Верхнеимбатск (63°9′17″ с. ш. 87°58′1″ в. д.; 93/109 км на с-з от п. Бахта/стационара “Мирное”), полученными из архивов ФГБУ “ВНИИГМИ-МЦД”.

В качестве опорного для расчёта климатических норм был использован период 1961-1980 гг. Все вычисления были выполнены с помощью языка статистической обработки данных R в среде Rstudio. Были использованы как самостоятельно написанные функции, так и готовые решения, такие как:

1)экосистема пакетов tidyverse, включающая в себя пакеты dplyr для “аккуратных” вычислений, ggplot2 для построения графиков и другие.
2)readxl для загрузки данных в формате xlsx.
3)plotly для построения интерактивных визуализаций при разведочном анализе данных.
4)VIM для визуализации количества пропущенных значений.
5)imputeTS для интерполяции пропущенных измерений.

Результаты

Тренды среднегодовой температуры и сумм осадков

Согласно проведённому анализу, на территории исследований наблюдается положительный тренд средней годовой приземной температуры воздуха с коэффициентом линейного тренда b=0.0483 °C/ 10 лет и годовой суммы осадков (b=0.995 мм/10 лет), однако статистически значимым является лишь тренд средней годовой температуры воздуха.

Показать код 
library(tidyverse)
library(ggpmisc)
library(patchwork)
Bakhta_monthly <- read.csv2("../data/cleaned/Bakhta_monthly_1966-1976_replaced_by_average_Bor_Verkhn.csv") |> 
  filter(!is.na(Date))

Bakhta_annualy <- Bakhta_monthly |> 
  filter(Year!= 2023) |> # Так как данные за 2023 год только до 31 мая
  group_by(Year) |> 
  summarise(Tavg = mean(Tavg),
            Pr = sum(Pr))

same_components_for_graph <- function(){
  list(
      stat_poly_line(),
      stat_poly_eq(use_label(c("eq"))),
      stat_poly_eq(use_label(c("R2", "P")), label.y = 0.85),
      theme_bw(),
      theme(text=element_text(family="sans", size = 10,
                              colour = "black" ),
            plot.caption = element_text(size = 11,
                                        hjust = 0.5,
                                        lineheight = 0.5),
            plot.title = element_text(size = 14,
                                        hjust = 0.5,
                                        lineheight = 0.5)
      )
      
  )
} # добавляет на график линию тренда и уравнение


# график среднегодовой температуры воздуха
temperature <- ggplot(Bakhta_annualy, aes(Year, Tavg))+
  geom_line()+
  labs(y = "Среднегодовая температура \n воздуха [\u00B0C]",
       x = "")+
  same_components_for_graph()

# график годовых сумм осадков
prec <- ggplot(Bakhta_annualy, aes(Year, Pr))+
  geom_col()+
  labs(y = "Годовая сумма осадков [мм]",
       x = "Год")+
  same_components_for_graph()

# сведенные графики
temperature / prec + plot_annotation(
  title = "Тренды среднегодовой температуры и сумм осадков", 
      caption = 
      "y = intercept+b*x  - уравнение линейной регрессии,
      где x - год,
      b - угол наклона регрессионной прямой,
      intercept - свободный член уравнения.
      R\u00B2 - коэффициент детерминации уравнения линейной регрессии
      P - p-уровень значимости") &
      theme(plot.caption = element_text(size = 11,
                                        hjust = 0.5,
                                        lineheight = 0.7))

Аномалии приземной температуры воздуха

Аномалии были рассчитаны как разность между средней температурой воздуха в определённый год или месяц и средней годовой/месячной температурой за выбранный опорный период (1961-1980).
На территории исследований наблюдается статистически значимый положительный тренд аномалий средней годовой приземной температуры воздуха с коэффициентом линейного тренда b=0.0483 °C/10 лет. Кроме того, значимы положительные тренды для следующих месяцев (в порядке убывания темппов роста): февраля (b=0.0941°C/10 лет), мая (b=0.0903°C/10 лет), марта (b=0.0857°C/10 лет), апреля (b=0.076°C/10 лет), июня (b=0.0537°C/10 лет), октября (b=0.0631°C/10 лет).

Показать код 
# средние значения за опорный период
base_annual <- Bakhta_monthly |> 
  filter(Year %in% c(1961:1980)) |> 
  summarise(Tavg_base = mean(Tavg),
            Pr_base = mean(Pr))
# среднегодовые аномалии
Bakhta_annual_anomaly <- Bakhta_monthly |>
  group_by(Year) |> 
  summarise(Tavg = mean(Tavg),
            Pr = mean(Pr),
            Sn = mean(Sn)) |> 
  mutate(Tavg_base = base_annual$Tavg_base,
         Pr_base = base_annual$Pr_base) |> 
  mutate(Tavg_anomaly = Tavg - Tavg_base,
         Pr_anomaly = Pr - Pr_base)

# среднемесячные за опорный период
base_monthly <- Bakhta_monthly |> 
  filter(Year %in% c(1961:1980)) |> 
  group_by(Month) |> 
  summarise(Tavg_base = mean(Tavg),
            Pr_base = mean(Pr))
# аномалии по месяцам
Bakhta_monthly_anomaly <- Bakhta_monthly |> 
  left_join(base_monthly, by = c("Month")) |> 
  mutate(Tavg_anomaly = Tavg - Tavg_base,
         Pr_anomaly = Pr - Pr_base)


#Графики

  geom_line()+
  labs(title = "",
       x = "",
       y = "Температурные аномалии, [\u00B0C]")+
  same_components_for_graph()

Bakhta_monthly_anomaly <- Bakhta_monthly_anomaly |> 
  mutate(Month = factor(Month, labels = c("Январь", "Февраль", "Март",
                                          "Апрель", "Май","Июнь","Июль",

                                          "Ноябрь", "Декабрь")))

# Отклонения  температуры воздуха по месяцам
graph_monthly_anomaly_temp <- ggplot(Bakhta_monthly_anomaly, aes(Year, Tavg_anomaly))+
  geom_line()+
  facet_wrap(.~Month, ncol = 3)+
  labs(title = "",
       x = "Год",
       y = "Температурные аномалии, [\u00B0C]")+

graph_annualy_anomaly_temp / graph_monthly_anomaly_temp +

  title = "Аномалии средней годовой и средних месячных

      R\u00B2 - коэффициент детерминации уравнения линейной регрессии
      P - p-уровень значимости") &

                                        hjust = 0.5,

Аномалии количества осадков

Статистически значимый положительный тренд аномалий количества осадков характерен только для февраля (b=0.191 мм/10 лет) и марта (b=0.319 мм/10 лет). Для остальных месяцев характерны незначимые положительные либо отсутствие тренда. Июль характеризуется статистически незначимым нисходящим трендом (b=-0.32 мм/10лет).
